# Локальный доступ только к сводке HASC

Дата решения: 2026-07-14.

Владелец явно разрешил HASC локальный доступ только к девяти общим числам
сводки дома. Это не разрешение на управление домом, доступ к списку устройств
или доступ из интернета.

## Что появилось

В версии `0.2.0` HASC добавляет одну защищённую страницу внутри Home
Assistant. Она умеет только отвечать на запрос чтения и возвращает девять
чисел из [безопасной сводки](read-only-home-summary.md).

Перед ответом HASC проверяет все условия:

1. Запрос пришёл из домашней сети или с самого Home Assistant.
2. Пользователь Home Assistant вошёл с отдельной ролью «только просмотр».
3. Этот пользователь не администратор, не системный пользователь и не состоит
   одновременно в какой-либо другой группе прав.
4. Конфигурация HASC по-прежнему находится в одном из двух безопасных
   режимов: «Только чтение» или «Проверка без изменений».

Если не выполнено хотя бы одно условие, HASC не выдаёт даже сводку.

## Чего этот доступ не даёт

- Он не показывает названия, идентификаторы, показания, историю или события.
- Он не принимает команды и не содержит кнопок управления.
- Эта отдельная страница не создаёт дополнительных устройств, сущностей, служб
  или автоматизаций. Девять строк HASC с общими числами описаны в
  [безопасной сводке](read-only-home-summary.md).
- Он не обращается к Node-RED, Climate, Automation или Smart Home Center.
- Он не отправляет данные наружу и не хранит пароль или ключ в HASC.
- Proxy и прямое выполнение остаются запрещены.

## Как включать безопасно

После обновления HASC сначала создаётся отдельный пользователь Home Assistant
для HASC. Он не должен быть администратором, не должен быть привязан к
человеку или присутствию и должен иметь только встроенную роль «только
просмотр».

Не передавайте пароль, адрес Home Assistant или ключ доступа в чат и не
кладите их в этот репозиторий. Когда HASC будет обновлён и такая роль будет
доступна в интерфейсе Home Assistant, следующий короткий шаг будет описан
отдельно. Если интерфейс не предлагает роль «только просмотр», остановитесь:
не редактируйте внутренние файлы Home Assistant вручную.
