# Решение: показать девять общих чисел в Home Assistant

Дата решения: 2026-07-15.

Владелец явно разрешил HASC показывать в Home Assistant только девять уже
разрешённых общих чисел без управления домом.

## Что разрешено

- Ровно девять строк HASC с числами из
  [безопасной сводки](read-only-home-summary.md).
- Числа берутся из той же короткой сводки, что диагностика и защищённая
  локальная страница.
- Обновление происходит локально внутри Home Assistant. Названия устройств,
  их номера, показания и история не сохраняются и не показываются.
- HASC не включает собственные девять строк в общий подсчёт дома.

## Что не разрешено

- Устройства HASC, кнопки, переключатели, службы, автоматизации и любые
  команды.
- Новые виды данных дома, имена комнат или устройств, показания датчиков,
  адреса, ключи и журналы.
- Proxy, прямое выполнение, изменение климата или бытовых сценариев.

## Как проверяется

Местные проверки требуют ровно девять строк с целыми неотрицательными
числами, отсутствие служб и удаление всех строк после удаления HASC. Отдельная
учебная проверка Home Assistant запускается только в пустой временной среде.
