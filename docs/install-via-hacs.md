# Установка HausmanHub через HACS

HausmanHub — это интеграция Home Assistant. HACS используется только для её
установки и обновления.

## Что потребуется

- Home Assistant Core 2026.6.4 или новее;
- уже установленный и настроенный HACS;
- доступ к Home Assistant с правами администратора.

## Первое добавление репозитория

1. Откройте **HACS** в боковом меню Home Assistant.
2. Откройте меню с тремя точками в правом верхнем углу.
3. Выберите **Пользовательские репозитории**.
4. В поле репозитория вставьте:
   `https://github.com/shumkiiv/hausmanhub_hacs`.
5. В поле типа выберите **Интеграция**.
6. Нажмите **Добавить**.

Эта последовательность соответствует
[официальной инструкции HACS для пользовательских репозиториев](https://www.hacs.xyz/docs/faq/custom_repositories/).

## Установка интеграции

1. В HACS найдите **HausmanHub**.
2. Откройте карточку и нажмите **Скачать**.
3. Выберите последнюю доступную версию.
4. После завершения загрузки перезапустите Home Assistant.
5. Откройте **Настройки → Устройства и службы**.
6. Нажмите **Добавить интеграцию**, найдите **HausmanHub** и откройте её.
7. Оставьте вариант **Обычный режим — рекомендуется** и завершите добавление.

После этого настройка HausmanHub открывается на карточке интеграции через
кнопку **Настроить**. Установка сама не выбирает устройства и не включает
управление климатом.

## Обновление

1. Откройте **HACS → HausmanHub**.
2. Если доступна новая версия, нажмите **Скачать**.
3. Выберите новую версию и дождитесь окончания загрузки.
4. Перезапустите Home Assistant, если HACS просит это сделать.

## Если HausmanHub не находится

- Проверьте, что пользовательский репозиторий добавлен именно как
  **Интеграция**.
- В HACS обновите сведения и повторно откройте поиск.
- Обновите страницу Home Assistant без использования кэша браузера.
- Убедитесь, что после загрузки существует каталог
  `custom_components/hausman_hub` и Home Assistant был перезапущен.
- Если интеграция всё равно не появляется, проверьте журнал Home Assistant на
  сообщения с доменом `hausman_hub`. Не публикуйте ключи, адрес дома или
  сведения об устройствах.
