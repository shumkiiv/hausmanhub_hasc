# Владелец разрешил показать девять общих чисел

Дата: 2026-07-15.

Владелец дал явное разрешение: «Разрешаю показывать в Home Assistant только
9 общих чисел HASC без управления».

Это разрешает ровно девять диагностических строк HASC: помещения, устройства,
все объекты, датчики, доступные, недоступные, неизвестные, без состояния и
отключённые. Каждая строка содержит только целое число из уже одобренной
сводки. Она не содержит название, номер, показание, историю или другую деталь
дома.

Решение не разрешает кнопки, переключатели, службы, автоматизации, Proxy,
прямое выполнение, связь с Node-RED или новые виды домашних данных. HASC не
считает свои девять строк частью дома.

Подробная граница для репозитория зафиксирована в
`docs/read-only-home-summary-display-decision.md`.
