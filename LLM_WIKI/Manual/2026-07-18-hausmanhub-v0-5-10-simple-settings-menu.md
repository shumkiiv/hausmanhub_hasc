# HausmanHub 0.5.10: простое меню настроек

Дата: 2026-07-18.

## Причина изменения

Прежний экран одновременно показывал режим общих сведений, страницу сводки,
интервал обновления, служебный переключатель, режим климатического контура,
адрес, комнату и переход к устройствам. Пользователю было трудно понять, какие
поля относятся к климату и какие из них вообще нужны.

## Новая структура

Первый экран параметров HausmanHub теперь содержит только выбор одного из четырёх
разделов:

1. комнаты и климатические устройства;
2. связь с существующим климатическим контуром;
3. общие сведения HausmanHub;
4. служебный тест переключателя, который явно обозначен как не относящийся к
   климату.

Общие сведения содержат только три своих параметра. Служебный тест содержит
только выключатель теста и выбор помощника Home Assistant. Подключение климата
сначала просит выбрать безопасный режим. Адрес показывается только для
проверки или пробного управления, а постоянный код комнаты — только для
пробного управления одной комнатой.

Основной мастер комнат и устройств сохраняет прежнюю отдельную проверку и
подтверждение. Названия его действий упрощены: отдельно обозначены найденное
устройство, ручное добавление, проверка комнаты, сохранение, расширенный JSON и
полное удаление списка.

## Сохранение и безопасность

Каждый раздел сохраняет только свои значения и переносит остальные настройки
только после их полной проверки. Если сохранённая конфигурация повреждена,
форма использует безопасные значения по умолчанию. Отключение связи с
климатическим контуром сразу удаляет сохранённый адрес и комнату.

Изменение не включает климатический контур, не меняет публичный Android
контракт, не настраивает живые устройства и не даёт новых прав на физические
команды.
