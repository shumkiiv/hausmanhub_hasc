# HausmanHub 0.3.1 — проверка имён и безопасного обновления

Дата: 2026-07-15.

## Что изменилось

- Для новой установки девять разрешённых строк HausmanHub получают понятный общий
  префикс во внутренних именах.
- Постоянные ключи строк не менялись. Поэтому уже установленная HausmanHub сохраняет
  свои текущие имена.
- Изолированная проверка теперь сначала создаёт временный вариант 0.3.0,
  затем заменяет только временную копию на 0.3.1 и проверяет оба случая:
  старые имена остаются, новая запись получает новые защищённые имена.
- Добавлен единый LF-формат текстовых файлов и статическая проверка префикса.
