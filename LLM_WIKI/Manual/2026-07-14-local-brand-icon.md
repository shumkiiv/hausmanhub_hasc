# Local brand icon

Date: 2026-07-14.

## Reason

The installed integration had no visual icon in Home Assistant. A local brand
asset is supported for custom integrations and keeps the image inside this
repository.

## Asset

`custom_components/hausman_hub/brand/icon.png` is an original 512 by 512 PNG
with transparent corners. Its mark is a house and a passive central hub: it
communicates observation and coordination only, not device control.

## Boundary

The image contains no text, credentials, identifiers, device command, or
runtime behavior. The accompanying local test checks that the expected square
PNG with an alpha channel remains present.
